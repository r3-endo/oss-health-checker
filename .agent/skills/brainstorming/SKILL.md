# ブレインストーミング

## 使う場面
- 新しい機能やプロジェクトの構想段階
- アイデアを設計・要件に落とし込みたいとき
- 複数アプローチを比較検討したいとき

## 進め方（簡易）
1. 一度に一つの質問だけ行う
2. 可能なら選択式の質問で選ばせる
3. 2〜3案の代替アプローチを出す
4. 最終的に結論と次の一手をまとめる

## 出力フォーマット（簡易）
- 目的/前提
- 主要な論点
- 代替案（2〜3）
- 結論
- 次アクション
