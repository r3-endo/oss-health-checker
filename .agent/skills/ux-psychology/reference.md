# UX心理学ガイド（要約）

出典: https://www.shokasonjuku.com/ux-psychology

## 効果・バイアス一覧（定義 + UX活用メモ）

- 美的ユーザビリティ効果: 見た目が美しいUIは使いやすく感じられる傾向。UX活用: 視覚品質の改善は学習コスト低減に直結するため、初期印象の画面で優先。
- アンカー効果: 先に提示された情報が後続の評価に影響。UX活用: 価格/比較情報の順序や初期値を設計する。
- バナー・ブラインドネス: 広告のような要素は無意識に無視されやすい。UX活用: 重要な操作は広告的な視覚表現を避ける。
- 認知負荷: 情報処理の負荷が高いと理解が落ちる。UX活用: 情報量と選択肢を抑え、段階的に提示する。
- 確証バイアス: 既存の信念に合う情報を優先する傾向。UX活用: 反対意見も見える配置にして意思決定を補助。
- 好奇心ギャップ: 情報不足が行動を促進する。UX活用: 見出しやティザーで続きを促す。
- 決断疲れ: 選択が続くと合理的判断が難しくなる。UX活用: 重要な決定を前半に集約し、既定値を用意。
- おとり効果: 比較対象を置くと選好が偏る。UX活用: プラン比較で意図した選択を支援。
- デフォルト効果: 既定値に留まる傾向。UX活用: 推奨設定をデフォルトにする。
- ドハティの閾値 (0.4秒の壁): 0.4秒超の待ちで離脱リスク。UX活用: 0.4秒以内のレスポンスか、即時フィードバックを出す。
- 共感ギャップ: 作り手がユーザー感情に共感できない。UX活用: ユーザーの感情状態を前提にフローを設計。
- 授かり効果: 所有しているものを過大評価。UX活用: 体験版/下書き保存で「所有感」を作る。
- 期待バイアス: 事前期待が体験評価に影響。UX活用: 期待水準を適切に設計し、過剰な約束は避ける。
- 親近性バイアス: 見慣れたものを好む。UX活用: 既存の慣習パターンに合わせる。
- 段階的要請: 小さな同意の後に大きな同意が得やすい。UX活用: ステップを小さくし同意を積み上げる。
- フレーミング効果: 提示の仕方で判断が変わる。UX活用: 成功率/失敗率の見せ方を使い分ける。
- ゲーミフィケーション: ゲーム要素で動機付け。UX活用: 進捗、報酬、バッジの設計。
- 目標勾配効果: ゴールに近いほど努力が加速。UX活用: 進捗バーや残りステップの表示。
- ハロー効果: 企業印象が評価に影響。UX活用: 信頼を示す証拠（実績、認証）を見せる。
- 観察効果: 監視されている意識で行動が変化。UX活用: 透明性あるフィードバックで品質向上を促す。
- 意図的な壁: あえて手間を増やし慎重な判断を促す。UX活用: 破壊的操作に確認ステップを入れる。
- 労働の錯覚: 裏側の努力が見えると価値が上がる。UX活用: 進行中アニメーションで処理感を演出。
- 損失回避: 得より損を避けたい傾向。UX活用: 失うリスクを明示し、保存/バックアップを強調。
- ナッジ効果: 非強制的に望ましい行動へ誘導。UX活用: 推奨ラベルや配置で自然に誘導。
- ピーク・エンドの法則: 最高潮と終わりが評価を決める。UX活用: 最後の体験（完了画面）を丁寧に作る。
- プライミング効果: 事前の刺激が後続行動に影響。UX活用: 事前の例示や文言で期待を整える。
- 段階的開示: 必要な情報を段階的に出す。UX活用: まず重要情報のみ見せ、詳細は展開。
- ピグマリオン効果: 期待が結果を引き寄せる。UX活用: 成果を期待させる文言設計。
- 誘導抵抗: 制約や強制で反発が生じる。UX活用: 選択肢を残して自由度を担保。
- 反応型オンボーディング: 行動に応じて手順を出す。UX活用: 初回操作や困りポイントで提示。
- 希少性効果: 希少性が価値を高める。UX活用: 在庫/限定感を適切に表示。
- 選択的注意: 目的に沿う情報だけを見る傾向。UX活用: 主要タスクを強調しノイズを削る。
- 系列位置効果: 最初と最後が記憶に残りやすい。UX活用: 重要情報を先頭/末尾に配置。
- スキューモーフィズム: 現実世界の見た目を模倣。UX活用: 初学者向けの理解を助ける。
- 社会的証明: 他者の行動が判断に影響。UX活用: レビュー数、利用者数の提示。
- サンクコスト効果: 投資済みコストで継続しやすい。UX活用: 進捗保存や再開導線。
- 調査バイアス: 調査結果の偏り。UX活用: 質問設計と母集団の偏りに注意。
- 誘惑の結びつけ: 好きな活動と結びつける。UX活用: 面倒な作業に報酬を組み合わせる。
- ユーザー歓喜効果: 驚きや喜びが体験価値を高める。UX活用: 小さな喜びの演出を入れる。
- 変動型報酬: 予測不能な報酬が継続性を高める。UX活用: ガチャ的な演出は倫理配慮の上で。
- ビジュアル・アンカー: 視覚的に目立つ要素で注意を集める。UX活用: 主要CTAの強調。
- 視覚的階層: 情報の優先順位を視覚で示す。UX活用: サイズ/色/余白の設計。
- ツァイガルニク効果: 未完了タスクが記憶に残る。UX活用: 続きの動線を残して再訪を促す。

## 使い方メモ

- 施策を考えるときは「ユーザーの心理状態」「状況」「文脈」に合わせて効果を選ぶ。
- 一つの効果に頼りすぎず、複数の観点で検証する（倫理配慮を含む）。
