# UX心理学ガイド

## 使う場面
- UI/UX設計や改善施策の整理
- フロントエンドの画面/導線レビュー
- UIテストやABテストの仮説作成

## 進め方
1. 目的とユーザー文脈（状況/感情/制約）を整理する
2. 適用する心理効果を選び、仮説を一文で記述する
3. UI上の具体変更点（文言/配置/順序/デフォルト値）を決める
4. 期待される行動変化と計測指標を定義する

## チェックリスト
- [ ] 認知負荷を下げる工夫がある
- [ ] 重要な行動が視覚的に目立つ
- [ ] 既定値や順序が意図に合っている
- [ ] 期待値の過剰な吊り上げがない
- [ ] 破壊的操作には意図的な壁がある
